Назначение программы, описание: простая консольная реализация классической игры "Крестики-нолики" для двух игроков на одном компьютере.

Особенности
Чистый текстовый интерфейс в терминале
Пошаговая игра для двух игроков
Проверка корректности ходов
Автоматическое определение победителя
Возможность сыграть несколько раз подряд
Подсказка с нумерацией клеток

Требования
Python 3.6 или выше

Установка и запуск
Клонируйте репозиторий:

bash
git clone 
cd TTT.txt
Запустите игру:

bash
python TTT.py

Пример ввода/вывода: (Как играть)
Игроки ходят по очереди

Первый игрок играет за "X", второй за "O"

Введите номер клетки (1-9) согласно схеме на экране:

text
 1 | 2 | 3
---|---|---
 4 | 5 | 6
---|---|---
 7 | 8 | 9
Для победы нужно выстроить 3 своих символа в ряд

Игра продолжается до победы или ничьи

Технические детали
Код написан на чистом Python без внешних зависимостей

Используется ООП подход с классом TTT

Валидация всех пользовательских вводов

Кроссплатформенность (работает на Windows, macOS, Linux)
